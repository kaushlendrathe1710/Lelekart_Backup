# LeLeKart E-commerce Platform API Documentation

## Authentication and User Management
- `/api/register` - Register a new user (POST)
- `/api/login` - Authenticate user with password (POST)
- `/api/request-otp` - Request OTP for authentication (POST)
- `/api/verify-otp` - Verify OTP for authentication (POST)
- `/api/logout` - Log out the current user (POST)
- `/api/user` - Get current user information (GET)
- `/api/users` - Get all users (Admin only) (GET)
- `/api/users/:id` - Update user information (PUT)
- `/api/users/:id` - Delete a user (Admin only) (DELETE)

## Seller Management
- `/api/sellers/pending` - Get pending seller applications (Admin only) (GET)
- `/api/sellers/rejected` - Get rejected seller applications (Admin only) (GET)
- `/api/sellers/approved` - Get approved sellers (Admin only) (GET)
- `/api/sellers/:id/approve` - Approve a seller application (Admin only) (POST)
- `/api/sellers/:id/reject` - Reject a seller application (Admin only) (POST)
- `/api/seller/status` - Get current seller's status (GET)

## Co-Admin Management
- `/api/co-admins` - Get all co-admins (Admin only) (GET)
- `/api/co-admins/:id` - Delete a co-admin (Admin only) (DELETE)
- `/api/co-admins/:id/permissions` - Update co-admin permissions (Admin only) (PUT)

## Product Management
- `/api/products` - Get products with filtering & pagination (GET)
- `/api/products` - Create a new product (Seller only) (POST)
- `/api/products/:id` - Get a specific product (GET)
- `/api/products/:id` - Update a product (Seller only) (PUT)
- `/api/products/:id` - Delete a product (Seller only) (DELETE)
- `/api/products/pending` - Get pending product approvals (Admin only) (GET)
- `/api/products/:id/approve` - Approve a product (Admin only) (POST)
- `/api/products/:id/reject` - Reject a product (Admin only) (POST)
- `/api/products/:id/complementary` - Get complementary products (GET)
- `/api/seller/products` - Get seller's products (Seller only) (GET)
- `/api/products-bulk-upload` - Bulk upload products (Seller only) (POST)
- `/api/categories` - Get all product categories (GET)

## Cart and Wishlist
- `/api/cart` - Get user's cart items (GET)
- `/api/cart` - Add item to cart (POST)
- `/api/cart/:id` - Update cart item quantity (PUT)
- `/api/cart/:id` - Remove item from cart (DELETE)
- `/api/wishlist` - Get user's wishlist items (GET)
- `/api/wishlist` - Add item to wishlist (POST)
- `/api/wishlist/:id` - Remove item from wishlist (DELETE)

## Order Management
- `/api/orders` - Create a new order (POST)
- `/api/orders` - Get user's orders (GET)
- `/api/orders/:id` - Get a specific order (GET)
- `/api/orders/:id/status` - Update order status (Seller/Admin only) (PUT)
- `/api/seller/orders` - Get seller's orders (Seller only) (GET)

## Payment Integration
- `/api/create-order` - Create Razorpay order (POST)
- `/api/verify-payment` - Verify Razorpay payment (POST)

## Address Management
- `/api/addresses` - Get user's addresses (GET)
- `/api/addresses` - Add a new address (POST)
- `/api/addresses/:id` - Update an address (PUT)
- `/api/addresses/:id` - Delete an address (DELETE)
- `/api/addresses/:id/set-default` - Set an address as default (POST)

## Smart Inventory & Price Management (AI-Powered)

- `/api/seller/inventory/price-optimization` - Get price optimization suggestions (Seller only) (GET)
- `/api/seller/inventory/stock-alerts` - Get stock alerts (Seller only) (GET)

- `/api/seller/ai-generated-content` - Request AI-generated content (Seller only) (POST)

## File Management
